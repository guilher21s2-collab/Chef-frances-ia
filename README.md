- ## 📌 Sobre o Projeto

Chatbot desenvolvido em Python utilizando Inteligência Artificial com LangChain e Groq.

O projeto atua como um especialista em culinária francesa, respondendo perguntas sobre pratos típicos, ingredientes, receitas, técnicas culinárias e história gastronômica da França.

O chatbot utiliza como base de conhecimento o conteúdo do site:

https://acozinhafrancesa.com.br/cozinha-francesa/

## 🛠️ Tecnologias Utilizadas

- Python
- Google Colab
- LangChain
- LangChain Groq
- Groq API
- WebBaseLoader
- BeautifulSoup4

---

## ⚙️ Funcionamento

O usuário faz perguntas no terminal do Google Colab ou em uma IDE Python, e o modelo de IA responde utilizando o conteúdo carregado do site informado.

Caso a pergunta não seja sobre culinária francesa, o chatbot informa que é especialista apenas nesse tema.

---

## ▶️ Como Executar

Instale as bibliotecas:

python
pip install langchain langchain-groq langchain-community beautifulsoup4

Depois execute o arquivo principal:

python main.py
💬 Exemplos de Perguntas
Quais são os pratos típicos da culinária francesa?
Me diga algo interessante sobre a culinária francesa.
🚪 Encerramento

Digite:
alura
para encerrar o chatbot.
